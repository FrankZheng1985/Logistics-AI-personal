# 背景音乐资源目录

请将以下背景音乐文件放置在此目录中：

## 需要的文件

| 文件名 | 用途 | 风格描述 |
|--------|------|----------|
| `corporate.mp3` | 商务企业风 | 稳重、专业、节奏适中 |
| `upbeat.mp3` | 活力动感 | 节奏明快、充满活力 |
| `warm.mp3` | 温馨亲和 | 柔和、温暖、建立信任 |
| `tech.mp3` | 科技感 | 电子、未来感、创新 |

## 音乐来源推荐

1. **免版税音乐网站**：
   - [Pixabay Music](https://pixabay.com/music/) - 完全免费
   - [Free Music Archive](https://freemusicarchive.org/) - CC协议
   - [Bensound](https://www.bensound.com/) - 免费商用（需署名）

2. **付费版权音乐**：
   - [Epidemic Sound](https://www.epidemicsound.com/)
   - [Artlist](https://artlist.io/)

## 音频格式要求

- 格式：MP3
- 采样率：44100Hz
- 比特率：128kbps 或更高
- 时长：建议 60秒以上（会自动循环）

## 注意事项

⚠️ 请确保使用的音乐具有合法的商用授权！
